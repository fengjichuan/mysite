Download DragSwipe

Requires Hammer.js and jQuery or Zepto.

For Zepto:

Download Dragswipe.zepto.js

Download zepto.hammer.js - a zepto plugin for Hammer.

For jQuery:

Download Dragswipe.jquery.js

Configure DragSwipe

Enable a DragSwipe carousel by calling $(el).dragswipe(options). For example:

$('#carousel').dragswipe({
width: 320,
current_page_element: '#current_page',
total_pages_element:'#total_pages'
});

Options are:

  • width: width of individual carousel element
  • current_page_element: DOM selector for element which should contain current page number
  • total_pages_element: DOM selector for element which should contain total page count
  • turn_threshold: % of page width that must be dragged for snap-to-next-page to trigger. default to 0.1 or 10%

Useful Methods:

  • $(el).nextPage() - go to next page
  • $(el).prevPage() - go to previous page
  • $(el).currentPage() - returns current page number
  • $(el).totalPages() - returns number of pages
  • $(el).gotoPage(pageNum) - go to page pageNum
